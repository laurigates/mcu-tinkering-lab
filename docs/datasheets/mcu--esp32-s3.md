# ESP32-S3 (Xtensa LX7)

**Used in:** Waveshare ESP32-S3-Zero

## Key Specs

| Parameter | Value |
|-----------|-------|
| Cores | Dual Xtensa 32-bit LX7 |
| Clock | Up to 240 MHz |
| SRAM | 512 KB |
| ROM | 384 KB |
| WiFi | 802.11 b/g/n |
| Bluetooth | BLE 5.0 |
| USB | USB OTG 1.1 |
| Flash | External, up to 16 MB (Octal SPI) |
| PSRAM | External, up to 16 MB (Octal SPI) |
| AI acceleration | Vector instructions for neural networks |
