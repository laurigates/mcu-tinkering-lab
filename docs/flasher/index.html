Setup instructions
  • Chrome or Edge required (Web Serial API). Firefox and Safari are not supported.
  • Android: Use Chrome 117+ with a USB-C OTG adapter. Grant USB permission when prompted.
  • ESP32-CAM (GPIO0 mode): Most boards lack onboard USB — connect an FTDI/CP2102 adapter. Hold GPIO0 LOW while pressing RESET to enter flash mode. Release GPIO0 after flashing begins.
  • ESP32-S3 (USB mode): Hold BOOT while plugging in USB, or hold BOOT then press and release RESET.
  • Heltec / standard DevKit boards: Connect via onboard USB. No special boot mode needed.
  • After flashing: WiFi credentials are not included in pre-built firmware. Use the serial console (Logs & Console in ESP Web Tools) or Improv WiFi (when supported) to configure your network.
