RoboCar Web Flasher

Flash firmware directly from your browser via USB

Your browser does not support Web Serial API. Please use Google Chrome or Microsoft Edge (version 89 or later) on desktop, or Chrome 117+ on Android with USB OTG.

Main Controller

Heltec WiFi LoRa 32 V1

  • Motor control (L298N)
  • PCA9685 LED/servo driver
  • OLED display, buzzer
  • WiFi OTA support

Camera Module

ESP32-CAM (AI Thinker)

  • OV2640 camera + AI vision
  • Claude / Ollama / Gemini backends
  • MQTT telemetry
  • WiFi OTA support
Setup instructions
  • Chrome or Edge required (Web Serial API). Firefox/Safari are not supported.
  • Android: Use Chrome 117+ with a USB-C OTG adapter. Grant USB permission when prompted.
  • ESP32-CAM: Most boards lack onboard USB. Connect an FTDI/CP2102 adapter. Hold GPIO0 while pressing RESET to enter flash mode. Release GPIO0 after flashing begins.
  • Heltec board: Connect via the onboard micro-USB port. No special boot mode needed.
  • After flashing: WiFi credentials are not included in pre-built firmware. Use the serial console (Logs & Console button in ESP Web Tools) or Improv WiFi (when available) to configure your network.
